# PokeQuiz
**Project Name:** PokeQuiz

**Public Database:** https://www.kaggle.com/datasets/guavocado/pokemon-stats-1025-pokemons/data

**Description:** PokeQuiz is an application that lets someone make a profile, set their favorite pokemon, create notes on them, submit feedback and test their knowledge on pokemon different pokemone triva based upon the stats. Then see their rankings upon other users. Along with that, users are able to look up profiles and different pokemon information that could help them on their way, like different egg groups.
